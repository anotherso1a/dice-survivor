# assets/ui/ — UI 素材

## 目录职责

存放 UI 界面使用的视觉素材，包括按钮、面板、图标等。

## 资源分类

| 子类 | 说明 |
|------|------|
| buttons/ | 按钮素材（正常/悬停/按下/禁用状态） |
| panels/ | 面板背景（九宫格拉伸素材） |
| icons/ | 功能图标（骰子/技能/遗物/元素图标） |
| bars/ | 进度条素材（血条/经验条） |

## 开发规范

- 按钮使用 NinePatchRect 实现可拉伸边框
- 图标尺寸统一：16×16 / 32×32 / 64×64
- 命名：`{type}_{variant}.png`，如 `btn_primary.png`、`icon_fire.png`
- UI 素材不需要关闭 Filter（与像素角色不同，UI 可以抗锯齿）
