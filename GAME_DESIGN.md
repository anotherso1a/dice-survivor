# 🎲 《骰子幸存者》Dice Survivor — 游戏设计文档

> **核心口号**：丢骰子打怪，骰子进战斗是 1 强，进赌局是 6 强，**怎么选都是博弈**。
>
> **灵感融合**：土豆兄弟（幸存者循环）+ Balatro（花式骰子构筑）+ 杀戮尖塔（遗物系统）+ 百家乐/吹牛/炸金花（休息站小游戏）
>
> **美术**：像素风（8-bit 复古调色板，32 色色板）
>
> **目标平台**：Steam（PC/Mac/Linux）→ 后期再考虑移动
>
> **引擎**：Godot 4.3+ / GDScript

---

## 一、核心反转机制（整个游戏的灵魂）

### 1.1 双轨价值观

| 场景 | 谁是爹 | 原因 |
|------|--------|------|
| **战斗** | **1** | 1 = 暴击 / 强化 / 特殊效果 |
| **战斗** | 2~6 | 普通伤害（5 略强、6 略强） |
| **百家乐/吹牛/炸金花** | **6** | 总和越大越赢、比 7 大、叫大数、凑顺子 |
| **炸金花** | **豹子/顺子** | 经典大点玩法 |

### 1.2 一颗骰子，两个身份

每颗骰子有 **两套点数规则**，切场景时自动切换：

```
【战斗模式】
  1 → ×3 暴击伤害 + 触发元素
  2 → 普通 ×1.0
  3 → 普通 ×1.0
  4 → 普通 ×1.0
  5 → 普通 ×1.1
  6 → 普通 ×1.2

【赌博模式】（休息站切到）
  1 → 1 点（垃圾）
  2 → 2 点（垃圾）
  3 → 3 点（普通）
  4 → 4 点（普通）
  5 → 5 点（好）
  6 → 6 点（顶级）
```

> 💡 **设计动机**：玩家在战斗里把"1"养成神器，到赌桌上就变成**最弱的那张牌**。**这才是真正的张力**。

## 二、博弈循环 (Strategy Loop)

```
[ 战斗 1~5 关 ]  → 把主骰改成"1 必暴击"流，强是强
       ↓
[ 休息站 1 ]    → 想玩百家乐？你的主骰现在全是 1...
                    要么不赌（保守），
                    要么换副骰去赌（风险），
                    要么卖血换金币（苟）
       ↓
[ 战斗 6~10 关 ] → 后悔了，骰子被赌神吃了
                    开始攒"分战场"：战斗骰 vs 赌博骰
       ↓
[ 休息站 2 ]    → 用副骰（主赌博用）赢了一波
       ↓
... 循环到 20~30 关
       ↓
[ BOSS ]        ← 多阶段，骰面机制特化
       ↓
[ 胜利 / 重玩 ]
```

### 三条路线

| 路线 | 玩法 | 优 | 劣 |
|------|------|----|----|
| **战斗流** | 拼命强化 1，赌局用金币买 | 战斗无脑通关 | 赌博赢不到好东西 |
| **赌博流** | 战斗凑合，赌局大杀四方 | 拿遗物/骰子/HP 多 | 战斗容易翻车 |
| **均衡流** | 骰子有战斗面、有赌博面，灵活切 | 灵活 | 啥都不顶尖 |

### "1 太多"的破局选项

| 选项 | 说明 |
|------|------|
| 切副骰 | 专门为赌局养一颗"全 6"骰 |
| 卖血 | 扣 5 HP 换 10 金币，**用命换入场券** |
| 苟 | 直接跳过这关赌局，奖励差但不掉血 |
| 吃诅咒 | 赌输了扣血也接受，因为赌赢能拿到稀世遗物 |

## 三、骰子体系 (Dice System)

### 3.1 骰子属性维度

| 维度 | 取值示例 | 作用 |
|------|---------|------|
| `面数 (sides)` | d4 / d6 / d8 / d12 / d20 | 骰子池构成 |
| `基础伤害 (power)` | 1 / 2 / 3 / 5 | 一点伤害 = 多少 HP |
| `冷却 (cooldown)` | 1s / 2s / 3s | 每隔多久自动摇一次 |
| `骰面 (faces)` | 普通 / 强化 / 属性 / 诅咒 | 摇到对应面触发什么 |
| `耐久 (durability)` | ∞ / 10 / 3 / 1 | 用 N 次碎裂（玻璃骰子） |
| `元素 (element)` | 无 / 火 / 冰 / 雷 / 毒 / 神 / 暗 | 元素反应 |

### 3.2 普通系骰子

| 骰子 | 战斗面 | 6 面 | 冷却 | 备注 |
|------|--------|------|------|------|
| **标准 d6** | 1 = ×3 暴击 | ×1.2 | 1.0s | 起步默认骰 |
| **灌铅 d6** | 1 = ×5 暴击 | ×1.5 | 2.0s | 1 更强，6 也比标准 6 强 |
| **玻璃 d6** | 1 = ×3 暴击 | ×4 | 0.8s | **10 耐久**用完碎 |
| **重 d6** | 1 = ×2 | ×1.0 | 1.0s | 都平均，没惊喜 |
| **穿刺 d6** | 1 = ×3 + 50% 破甲 | ×1.2 | 1.0s | 打坦克 |
| **幸运 d6** | 1 = ×3 + 30% 升 6 | ×1.0 | 1.0s | 赌狗专属 |

### 3.3 属性系骰子（带元素）

| 骰子 | 元素 | 1 面 | 6 面 |
|------|------|------|------|
| **火焰 d6** | 🔥 火 | ×3 暴击 + 3 燃烧 | ×1.0 |
| **冰霜 d6** | ❄️ 冰 | ×3 暴击 + 冻结 2s | ×1.0 |
| **雷霆 d6** | ⚡ 雷 | ×3 + 弹射 2 目标 | ×1.0 |
| **毒 d6** | ☠️ 毒 | ×3 + 中毒 5s | ×1.0 |
| **神圣 d6** | ✨ 神 | ×3 + 自己回 2 HP | ×1.0 |
| **暗影 d6** | 🌑 暗 | ×3 + 偷 1 攻击 | ×1.0 |

### 3.4 赌博专用骰子（均衡流玩家也能爽）

| 赌博骰 | 特点 | 战斗用途 |
|--------|------|---------|
| **黄金 d6** | 6 个面都是 6 | 战斗持续弱攻，赌桌神器 |
| **老千 d6** | 5/6 = 5 伤害（战斗也算暴击） | 战斗/赌局双修 |
| **暗骰 d6** | 数字看不见，揭牌时显示 | 吹牛神器 |
| **叠叠 d6** | 同时摇 2 颗 | 战斗多面，赌局凑对子 |
| **JJ d6** | 2 颗 d6 同点数触发双倍 | 炸金花专用 |
| **迷之 d6** | 永远 4~6 之间 | 战斗很弱，赌局中等偏强 |

### 3.5 模式切换机制

- 进战斗 → **自动切到战斗面**
- 进休息站 → **自动切到赌博面**
- 玩家可以**手动锁定某颗骰子的模式**（要花金币或技能）
- **遗物"双面人"**：可以**一颗骰子同时算两套点数**

## 四、骰面 (Faces) 详解

每张骰面是**带触发条件**的效果卡，分四类：

| 类别 | 示例 | 触发条件 |
|------|------|---------|
| **普通面** | 1 点伤害 | 摇到任意值（默认） |
| **强化面** | 3 点伤害 | 摇到 ≥ 5（普通骰），或 ≥ 7（d8），或被技能/遗物改写 |
| **属性面** | 1 伤害 + 2 燃烧 | 由属性骰子自然产生，或由"附魔"技能改写 |
| **诅咒面** | 1 伤害 + 自己掉 1 HP | 出现概率低，伤害爆炸 |

> 💡 **设计要点**：骰面是"可热插拔"的对象。技能作用的对象不是"骰子"，而是"骰子上的某一面"。

## 五、技能体系 (Skill System)

### 5.1 技能作用对象

| 技能类型 | 作用对象 | 例子 | 稀有度 |
|---------|---------|------|--------|
| **新增骰面** | 给骰子加一面 | "给主骰加一面【暴击：×2 伤害 + 回 1 HP】" | 稀有 |
| **改写骰面** | 改某面的数值/效果 | "把摇到 1 的面改成【重击：×3 伤害】" | 稀有 |
| **删除骰面** | 去掉某面 | "删除摇到 1 的面"（剩下 5 面） | 普通 |
| **附魔骰面** | 给某面加属性/效果 | "给所有普通面附上【火焰：+1 燃烧】" | 稀有 |
| **强化判定阈值** | 改"什么算强化" | "摇到 ≥ 4 就算强化攻击" | 史诗 |
| **复制骰子** | 复制一个骰子 | "复制主骰，伤害 -30%" | 史诗 |
| **改变冷却** | 改摇骰节奏 | "主骰冷却 -0.3s" | 普通 |
| **骰子融合** | 合成新骰子 | "火+雷 = 雷火 d6，6 = 范围燃烧" | 史诗 |

### 5.2 战斗向技能（强化 1）

| 技能 | 效果 | 稀有度 |
|------|------|--------|
| **1 锁定** | 每摇 3 次必出一次 1 | 白 |
| **1 强化** | 1 的暴击 ×3 → ×5 | 蓝 |
| **1 转冰** | 1 触发时附冻结 2s | 蓝 |
| **1 转火** | 1 触发时附 3 燃烧 | 蓝 |
| **1 转雷** | 1 触发时附弹射 2 目标 | 蓝 |
| **1 范围** | 1 触发时**全体**敌人受暴击 | 紫 |
| **1 = 全场** | 整个 d6 全是 1（锁死 6 面暴击） | 金 |

### 5.3 赌博向技能（强化 6）

| 技能 | 效果 | 稀有度 |
|------|------|--------|
| **6 锁定** | 每摇 3 次必出一次 6 | 白 |
| **赌徒之眼** | 百家乐看到 1 张底牌 | 蓝 |
| **鬼牌** | 某次骰子结果 ×2 | 蓝 |
| **暗牌** | 你摇的骰子**不显示**给对方看 | 蓝 |
| **连庄** | 连续赢 3 把后，奖励 ×3 | 紫 |
| **梭哈** | 单局赌注 ×5，输赢都 ×5 | 紫 |
| **天选之人** | 下一把**必赢**（一次性） | 金 |

### 5.4 双面技能（同时影响战斗和赌局）

| 技能 | 效果 | 稀有度 |
|------|------|--------|
| **反转** | 1 和 6 数值**互换**（1 强 6 强） | 紫 |
| **中和** | 摇到 1 时**也按赌博面**算 1 点 | 蓝 |
| **平衡** | 1 和 6 **都按平均值**（3.5）算 | 白 |
| **赌神附体** | 一颗骰子**同时 1 强和 6 强** | 金 |

### 5.5 升级三选一（战斗后）

每场战斗结束从 **3 张技能卡** 里选 1（Balatro 风格），白→蓝→紫概率随关卡爬升。

## 六、合成体系 (Crafting / Fusion)

### 6.1 基础合成公式

| 公式 | 输入 | 输出 | 效果 |
|------|------|------|------|
| 火 + 雷 | 🔥d6 + ⚡d6 | ⚡🔥 雷火 d6 | 1 = 范围 3 目标 + 2 燃烧 |
| 冰 + 水 | ❄️d6 + 💧d6 | 🌊 冰潮 d8 | 1 = 冻结 2s + 溅射 |
| 火 + 毒 | 🔥d6 + ☠️d6 | ☠️🔥 剧毒火 d6 | 1 = 5 燃烧 + 中毒 |
| 神 + 暗 | ✨d6 + 🌑d6 | 💫 虚空 d6 | 1 = 8 真实伤害（无视抗性） |
| 任意 + 玻璃 | Xd6 + 🪟d6 | 💎 棱镜 d6 | X 属性 + 玻璃 ×3 伤害，3 耐久 |
| 普通 + 元素 | d6 + 元素 d6 | 元素 d8 | 升级到 d8，伤害 +1 |
| 同元素 × 3 | 3 个同元素 d6 | 三相 X d6 | 每摇必触发一次元素效果 |

### 6.2 战斗-赌博 跨界合成（重点 ⭐）

| 公式 | 输出 | 效果 |
|------|------|------|
| 战斗 d6 + 黄金 d6 | **双面 d8** | 1 暴击（战斗），6 暴击（赌局） |
| 战斗 d6 + JJ d6 | **双 d6 套** | 一次摇 2 颗（战斗双伤害，赌局凑对子） |
| 战斗 d6 + 暗骰 d6 | **心机 d6** | 战斗时 1 暴击，赌局时不显示 |
| 火 d6 + 黄金 d6 | **炼金 d6** | 1 = 火暴击，6 = 火 + 金币翻倍 |
| 冰 d6 + 暗骰 d6 | **欺骗 d6** | 1 = 冰暴击，6 = 冻结对方（吹牛用） |
| 雷 d6 + 黄金 d6 | **财神 d6** | 1 = 雷暴击，6 = 抢对方金币 |

> 💡 **设计意图**：玩家想"双修"就得**用好骰位**——一颗给战斗，一颗给赌局，**合成的代价是其中一面变平庸**。

### 6.3 合成在哪儿做？

- **休息站**给一个合成台（每休息站只能用 1 次）
- **BOSS 前** 必给一次免费合成
- **遗物**可以解锁"战斗内合成"（精英怪掉落）

## 八、休息站：小游戏 (Minigames)

### 8.1 基础机制

- 进入休息站必玩 1 个小游戏
- 赢了得**金币 / 遗物 / 永久回血 / 骰子升级券**
- 输了扣 **HP** 或丢 **金币**
- 难度可选：简单（赢面大奖励差）/ 普通 / 地狱（赢面小奖励爆炸）

### 8.2 小游戏清单

| 名称 | 玩法 | 奖励池 |
|------|------|--------|
| **比大小**（百家乐简化） | 2 骰子，猜总和 > 7 / < 7 / = 7 | 金币 / 随机遗物 |
| **吹牛**（Liar's Dice） | 大家各自摇，暗牌，喊"至少 3 个 4" | 遗物 / 骰面改写 |
| **炸金花**（3 颗骰子版） | 摇 3 颗，比豹子/顺子/同花/对子/大点 | 骰子 / 金币 / HP |
| **21 点** | 凑 21 不过爆 | 金币 |
| **七上八下** | 摇到 7+ 算赢 | HP / 经验 |
| **骰子轮盘** | 押单点数（1~6），摇中给 5× 奖励 | 史诗遗物 |
| **双骰赌** | 2 骰子 vs 庄家比大小 | HP 池 |

## 九、战斗系统 (Combat)

### 9.1 战斗自动进行

- 玩家**站着不动**（或可轻微位移），骰子**自动摇、自动攻击**最近的敌人
- 玩家干预手段：拾取物、技能释放、闪避（看时机按）

### 9.2 敌人设计

| 类型 | 行为 | 抗性 |
|------|------|------|
| **小怪 1**（"6 之民"） | 摇到 6 才有伤害；玩家 1 → 一刀秒 | 无 |
| **小怪 2**（"6 党徒"） | 6 触发护盾（很硬），非 6 不破防 | 无 |
| **射手** | 远程 | 无 |
| **坦克** | 慢速 | 高物理抗 |
| **自爆** | 接近爆 | 死后范围伤害 |
| **精英** | 多技能 | 抗一种元素 |
| **BOSS 1**（"反 1 之王"） | 玩家 1 暴击时，它获得能量；逼迫玩家**别一直 1** | — |
| **BOSS 2**（"豹子女王"） | 3 个骰子豹子时无敌 | — |

## 十、像素风美术 (Pixel Art)

- 角色：**16×16** 或 **32×32** 像素小人
- 骰子：每面贴图不同，**摇出 1 时闪金光**（战斗），**摇出 6 时闪金光**（赌局）
- 敌人：参考土豆兄弟调色板，2~4 帧动画
- UI：复古 8-bit 字体（`Press Start 2P` or `PixelOperator`）
- 特效：**像素粒子**（爆字、爆点、火焰帧）
- 调色：限制 32 色或 64 色色板（**PICO-8** 风格），统一观感

> 💡 美术可以用程序化像素生成（Aseprite + 调色板脚本），单角色精灵 ~50 张

## 十三、设计哲学（做决定时的尺子）

1. **骰子是中心** — 所有系统都围着骰子转（技能/遗物/合成/小游戏都是"骰子效果器"）
2. **看得见的概率** — 骰子要**真的在屏幕里滚**，不能黑箱结算
3. **风险回报** — 玻璃骰、献祭骰、诅咒面都是"赌狗最爱"
4. **花样缝合** — 每个新内容都能挂到现有骰子上，不能加新按钮
5. **像素克制** — 32 色色板，复古不掉档次
6. **MVP 优先** — 先能摇骰子打怪，**再**加花活
7. **1 是爹** — 战斗时 1 越强，赌局越菜——**真正的核心张力**
