### 技术决策
- 所有 UI 面板均为 CanvasLayer，互不干扰渲染层级
- 面板间互斥：打开一个面板时自动关闭其他面板
- 通信方式：UI 面板 → EventBus 信号订阅（完全解耦）
- 配色方案：暗海军蓝背景 (#1a1a2e)，金色强调 (#f0c040)，适配像素风
- 使用 gl_compatibility 渲染器，确保所有 UI 兼容

### 注意事项
- TSCN 文件使用临时 UID，首次在 Godot 编辑器中打开时可能需要重新关联脚本（通过检查器拖入 .gd 文件）
- 遗物和骰子数据依赖 RunState 和 EventBus 信号，当前 MVP 阶段遗物列表为空是正常的
