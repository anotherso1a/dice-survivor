### 场景动画效果（已用 Tween 实现）

| 名称 | 效果 | 实现方式 |
|------|------|----------|
| 骰盅入场 | 从上方弹跳落下 + 淡入 | `tween_property(y) + modulate.a` |
| 摇晃 | 左右震荡循环 + 着色器运动模糊 | `tween + loops() + cup_shake.gdshader` |
| 落地"啪" | 垂直挤压回弹 | `tween_property(scale)` → 1.08→0.92→1.0 |
| 揭牌 | 骰盅飞起消失 + 骰子从小弹大出现 + 发光衰减 | `tween_property(y/scale/flash)` |
| 结果弹窗 | 从中心弹大出现 + 弹小消失 | `tween_property(scale)` BACK ease |

### 场景动画效果

| 名称 | 效果 | 实现方式 |
|------|------|----------|
| 玩家行走 | 4帧 spritesheet 循环 + 上下弹跳 | `_physics_process` 帧切换 + `AtlasTexture` |
| 玩家朝向 | 左右翻转 | `sprite.flip_h` |

| 路灯 | 呼吸光晕 | `lamp_glow.gdshader` + `TIME` |
| 石板路 | 程序化砖块纹理 | `village_floor.gdshader` |

## 四、像素尺寸规范

| 类别 | 建议尺寸 | 说明 |
|------|---------|------|
| 角色（玩家/NPC） | 24×32 | 2:3 比例，侧视角 |
| 骰子 | 32×32 | 正方形，容纳 1-6 点 |
| 骰盅 | 48×48 | 正方形，容纳盖子+杯身 |
| UI 图标 | 16×16 | 金币、箭头等小图标 |
| 路灯/装饰 | 16×48 | 纵向细长物体 |
| 场景 tile | 32×32 | 建筑墙体标准 tile |
| 背景 | 128×64 | 天空/远景 tile |

> 所有尺寸均符合 PICO-8 风格（小尺寸、像素化），在 1280×720 视口中约放大 2-3 倍。
